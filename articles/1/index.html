如何在Windows上安装OpenSSL

更新时间：2018/10/09 19:00:00 GMT+8

准备

  • 下载并解压OpenSSL的源代码（例如：1.1.1）。
  • 下载并安装Strawberry Perl（例如：5.28.0.1 x64）。
  • 下载并安装Visual Studio 2017（例如：Community）。
  • 下载并安装NASM（例如：2.13.03 x64）。你可能需要手动将NASM的安装路径添加到PATH。

安装

  • 在开始菜单以管理员身份运行VS2017开发者命令行工具。
  • 将当前路径更改到OpenSSL源代码解压到的路径。
  • 依次运行下列命令： 
    perl Configure VC-WIN32
nmake
nmake test
nmake install
  • 将"C:\Program Files (x86)\OpenSSL\bin"添加到PATH。

问题与解决方案

  • 运行"perl Configure VC-WIN32"时失败。尝试用"perl Configure VC-WIN32 no-asm"替代运行。
